¿Cómo se calcula?

  • De 0kg a 10kg, se calcula 100cc por cada kilo
  • Se suman 50cc por cada kilo de peso por arriba de 10kg, hasta 20kg
  • De 20kg para arriba, se suman 20cc por cada kilo adicional hasta los 30kg
  • En caso de superar los 30kg, se usa el método de superficie corporal, el cual usa una fórmula: Superficie corporal = ( (peso * 4) + 7) / (peso + 90)
